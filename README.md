# Analisi-dei-dati-e-big-data
Nel presente elaborato sono stati sviluppati tre progetti finalizzati all’applicazione di diverse tecnologie per la gestione e l’analisi dei dati.
In particolare:

•	Il primo progetto riguarda la realizzazione di un database relazionale per la gestione di un sistema universitario.  In questo sistema sono stati modellati studenti, docenti, corsi ed esami, definendo le relative entità e le relazioni tra di esse.  Successivamente è stato popolato il database con dati di esempio e, mediante il linguaggio Python, sono state implementate diverse query per l’estrazione delle informazioni richieste, come:

⁃	la visualizzazione degli esami sostenuti da uno studente comprensivi di voto e data;

⁃	il numero di studenti iscritti a ciascun corso;

⁃	la media dei voti ottenuti per un dato esame.

•	Il secondo progetto prevedeva la creazione di un Knowledge Graph in ambito sanitario mediante Neo4j.  Dopo aver collegato le diverse entità, quali pazienti, medici, visite, diagnosi, esami e farmaci, sono state definite le relazioni che descrivono tali collegamenti, rappresentando il percorso clinico effettivo dei pazienti. Una volta generato un grafo con dati simulati, sono stati sviluppati script in Python per interrogare il sistema e, tra le query, identificare:

⁃	 pazienti accomunati dalle stesse diagnosi e terapie farmacologiche;

⁃	 esami associati a specifiche patologie;

⁃	 i medici che hanno preso in carico pazienti affetti dalla stessa diagnosi.

Il terzo progetto riguarda la realizzazione di un indice su Elasticsearch dedicato alle recensioni dei film.  In una prima fase è stato creato l’indice e ne è stata definita la struttura, successivamente sono state inserite diverse recensioni dei film, comprensive delle principali informazioni descrittive degli utenti. Una volta popolato l’indice, mediante il linguaggio Python, sono state sviluppate ed eseguite diverse query volte ad interrogare i dati, effettuare ricerche mirate, applicare filtri ed estrarre informazioni specifiche dalle recensioni.
